# Diagrama de Casos de Uso – Projeto Avalia+

Esta pasta contém o **Diagrama de Casos de Uso** do projeto Avalia+.

O diagrama de casos de uso representa as interações entre os usuários e o sistema, evidenciando as principais funcionalidades disponíveis na plataforma.

---

## Objetivo do Diagrama de Casos de Uso

O diagrama tem como objetivo identificar e representar as funcionalidades do sistema a partir da perspectiva dos usuários, auxiliando no entendimento dos requisitos e no planejamento do desenvolvimento.
